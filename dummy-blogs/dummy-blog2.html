Introduction to for-loop

30 September

For loop is very common in programming. It can be used in programming for repeating javascript statements.

For loop, however, makes programming easier by avoiding writing the same code multiple times. We can specify initializer, condition, increment, or decrement.

the initializer we define a variable then we use it in condition to control the number of loops. After the condition, we can increment or decrement the variable we defined.

for ([initialization]; [condition]; [final-expression]) statement

initialization An expression or variable declaration evaluated once before the loop begins which typically is used to initialize a counter variable.

condition An expression that is evaluated before each loop iteration and if this expression evaluates to true the loop or statement is executed. This conditional test is optional. Note: If omitted, the condition always evaluates to true.

final-expression This is evaluated at the end of each loop iteration and occurs before the next evaluation of condition. Generally used to update or increment the counter or initialization variable.

statement A statement is line/lines of code that is executed as long as the condition evaluates to true. We can use block statement {( ... )} to for multiple lines of code

Example,

for (let i = 0; i < 9; i++) { //statements }

All the three parameters
([initialization]; [condition]; [final-expression])
are optional, example:

for (;;){
//statements
}

Above is a valid for-loop but it will be an infinite loop But we can do something like this with the same example,

for(;;){
if(/condition/){
break;
}
}

Other loops are while, do-while.
